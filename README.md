   * 能够将图片批量合并为PDF文件，支持小说模式（双面左至右，右至左排版，自动过滤横向图片）。<br/>
   * 支持绝大多数常见图片格式，支持webp图片。
   * 默认排序方式会根据文件名中的数字排序，若无数字的文件名会按照默认string排序方式排序。可在Config.lua文件中自定义排序方式。
